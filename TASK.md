创建一个安全生产主题的"大家来找茬"网页游戏，所有代码放在一个 index.html 文件中。

要求：
1. **纯前端**：HTML + CSS + JS，不依赖任何外部框架或库
2. **游戏机制**：
   - 3个关卡：施工现场、工厂车间、办公室
   - 每关两张图：安全示范图 vs 隐患图（用 Canvas 绘制）
   - 每关有5处安全隐患差异
   - 点击隐患图的差异处得分
   - 60秒倒计时
   - 有提示功能（高亮一个未找到的差异）
   - 有进度条和计分系统
3. **画风**：简洁的矢量/平面风格，用 Canvas 2D API 绘制场景
4. **每个差异要有中文描述**，找到后显示文字说明
5. **过关后显示安全知识小贴士**
6. **响应式设计**，手机也能玩
7. **界面美观**：暗色主题，渐变背景，动画效果
8. 开始界面有游戏说明

Canvas 绘制的场景元素要足够清晰，让人能看出差异。差异类型参考：
- 场景一（施工现场）：未戴安全帽、高空未系安全带、吸烟、电缆乱放、临边无护栏
- 场景二（工厂车间）：操作旋转设备戴手套、电箱门未关、消防通道堵塞、化学品无标识、灭火器过期
- 场景三（办公室）：排插串联堆放、文件柜未固定、消防通道堆物、插座超载、饮水机旁电线凌乱

把完整的 HTML 文件写到 index.html。

当完成后，运行:
openclaw system event --text "Done: 安全隐患找茬游戏已构建完成" --mode now
