# ⚠️ 安全隐患大家来找茬

一个安全生产主题的"找不同"网页游戏，通过互动方式提升安全意识。

## 🎮 游戏介绍

在施工现场、工厂车间、办公室等常见场景中，找出隐藏的安全隐患差异！

### 📌 游戏规则
- 每关有 **5处** 安全隐患需要找出
- 限时 **60秒**
- 点击右侧"隐患图"中的差异处即可得分
- 提示功能每次扣 **50分**

### 🏭 三大场景

| 关卡 | 场景 | 隐患类型 |
|------|------|---------|
| 1 | 施工现场 | 未戴安全帽、高空未系安全带、吸烟、电缆乱放、临边无护栏 |
| 2 | 工厂车间 | 旋转设备戴手套、电箱未关、消防通道堵塞、化学品无标识、灭火器过期 |
| 3 | 办公室 | 排插串联、文件柜未固定、通道堵塞、插座超载、电线凌乱 |

## 🚀 在线游玩

直接打开 `index.html` 即可开始游戏，无需安装任何依赖。

## 🛠️ 技术栈

- 纯前端：HTML + CSS + JavaScript
- 使用 Canvas 2D API 绘制场景
- 响应式设计，支持手机和电脑
- 零外部依赖
